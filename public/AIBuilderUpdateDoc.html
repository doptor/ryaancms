PHASE 1 Foundation — Engine, Pipeline & Core UI ✅ Complete

Engine Core (src/lib/engine/)

  • prompt-analyzer.ts — NLP-based prompt parsing: extracts project type, modules, features, pages, collections, roles, style preferences. Detects build target (website/application/plugin/combos).
  • component-registry.ts — Central type definitions: AppConfig, PageConfig, ComponentConfig, CollectionConfig, ApiEndpoint, FieldConfig, Role. 30+ component types registered.
  • orchestrator.ts — 10-agent autonomous pipeline with 17-step full build sequence. Smart Step Detection for micro-edits (3-5 steps). PipelineState management with agent logs.
  • database-generator.ts — Auto-generates Prisma schema + raw SQL from collections. Handles RLS, soft-delete, audit fields, relations, tenant isolation.
  • security-validator.ts — Scores project security (0-100). Checks: RLS, auth, input validation, rate limiting, CORS, SQL injection, XSS.
  • rbac-generator.ts — Generates RBAC middleware, policies, and permission matrices from role definitions.
  • theme-generator.ts — Generates CSS variables, Tailwind config, and component theme tokens from style preferences.
  • section-factory.ts — Maps component types to rendered UI sections in the preview.
  • module-templates.ts — Module scaffolding templates for auth, CRUD, media, payments, notifications, etc.
  • docs-generator.ts — Auto-generates 5 documentation files: README.md, INSTALL.md, API.md, DB_SCHEMA.md, SECURITY.md.
  • test-generator.ts — Generates automated test suites: Auth tests, CRUD tests, Security tests, Multi-tenant isolation tests.
  • index.ts — Barrel export for all engine modules.
