# RyaanCMS

**RyaanCMS** is an open-source, AI-native **Revenue Engine** and headless-hybrid CMS — a modern alternative to WordPress, designed to integrate content management with a marketplace and automation tools.

## Features

- 🧠 **AI-Native** — Built-in AI integration for content generation, SEO, and automation
- 🛒 **Marketplace** — Browse and install plugins, templates, and AI tools
- 🔌 **Plugin System** — Extensible architecture with .rypkg package support
- 🎨 **Schema Builder** — Visual database schema editor
- 🔐 **Authentication** — Secure user auth with email sign-up, sign-in, password recovery, and profile management
- 📊 **Dashboard** — Real-time analytics and content management overview
- 🌙 **Dark Mode** — Full dark/light theme support
- 📱 **Responsive** — Mobile-first, responsive design

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Lovable Cloud (Supabase)
- **AI:** Lovable AI Gateway

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd ryaancms

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Project Structure

```
src/
├── components/       # Reusable UI components
├── contexts/         # React context providers (Auth, etc.)
├── hooks/            # Custom React hooks
├── integrations/     # Backend integrations
├── pages/            # Route pages
└── lib/              # Utility functions
```
